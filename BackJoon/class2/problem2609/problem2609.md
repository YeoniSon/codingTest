# 최대공약수와 최소공배수
# 문제
> 두 개의 자연수를 입력받아 최대 공약수와 최소 공배수를 출력하는 프로그램을 작성하시오.

# 입력
첫째 줄에는 두 개의 자연수가 주어진다. 이 둘은 10,000이하의 자연수이며 사이에 한 칸의 공백이 주어진다.

# 출력
첫째 줄에는 입력으로 주어진 두 수의 최대공약수를, 둘째 줄에는 입력으로 주어진 두 수의 최소 공배수를 출력한다.

# 예제 입력 1
```
24 18
```
# 예제 출력 1
```
6
72
```

# 해결방안
- 최대공약수와 최소공배수를 구하는 함수를 따로 해서 작성하도록 한다.
- 최대공약수는 공약수중 최대를 구하는 값이고
- 최소공배수는 공배수중 최소를 구하는 값이다.
- 최대공약수는 GCD 함수로, 최소공배수는 LCM 함수로 나타낸다
- 최대공약수는 재귀함수를 이용해서 구한다.
- 공약수는 b가 0이라면 최대공약수가 a가 되고, 그렇지 않으면 b와 a % b의 최대공약수를 구한다.
- 최소 공배수는 두 값을 곱한후 최소 공약수를 나눠주면 된다.
