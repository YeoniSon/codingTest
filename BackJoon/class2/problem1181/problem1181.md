# 단어 정렬
# 문제
> 알파벳 소문자로 이루어진 N개의 단어가 들어오면 아래와 같은 조건에 따라 정렬하는 프로그램을 작성하시오.
>
> 1. 길이가 짧은 것부터
> 2. 길이가 같으면 사전 순으로
>
> 단, 중복된 단어는 하나만 남기고 제거해야 한다.
> 

# 입력
첫째 줄에 단어의 개수 N이 주어진다. (1 ≤ N ≤ 20,000) 둘째 줄부터 N개의 줄에 걸쳐 알파벳 소문자로 이루어진 단어가 한 줄에 하나씩 주어진다. 주어지는 문자열의 길이는 50을 넘지 않는다.

# 출력
조건에 따라 정렬하여 단어들을 출력한다.

# 예제 입력 1
```
13
but
i
wont
hesitate
no
more
no
more
it
cannot
wait
im
yours
```
# 예제 출력 1
```
i
im
it
no
but
more
wait
wont
yours
cannot
hesitate
```
# 해결방안
- Arrays.sort()함수를 사용한다. 사용시에 new Comparator<String>()를 이용한다.
- compare(String s1, s2)를 오버라이드를 사용하여 작성할 수 있다.
- 주어진 값들의 길이가 같을 경우에는 2가지 값을 비교해서 정렬하고,
- 같지 않을 경우에는 길이에 따라 정렬한다.
- 공통되는 값은 출력하지 않기 위해
- !arr[i].equals(arr[i-1])을 이용하여 작성한다.
- 런타임 에러를 막기 위해 StringBuilder을 이용하여야 한다.
